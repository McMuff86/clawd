# Südmetall VARIO STAR® TOP SPEED

![Südmetall Vario Star TopSpeed](suedmetall-vario-star-topspeed.jpg)

## Produktmerkmale

| Feature | Beschreibung |
|---------|--------------|
| **Kugellager** | Optional erhältlich (GK4) |
| **Schnellmontage** | "Top Speed" durch Bajonett-Verschluss |
| **Unterkonstruktion** | Metall = hohe Zuverlässigkeit |
| **Hochhaltefeder** | Integriert, perfekte Position |
| **Lagerung** | Fest/drehbar gelagert = kein Wackeln |
| **Optik** | Keine Innensechskant-Schraube bei GK3 |
| **Rosetten** | Ø 55mm – überdeckt 50mm + 52mm |

## Montage-Komponenten

- Spezialstift
- M4 Gewindeschrauben
- 7mm Stütznoppen
- Metallunterkonstruktion
- Integrierte Hochhaltefeder
- Bajonett-ähnlicher Verschluss

## Anwendung

- **Renovierung:** Ø 55mm Rosetten überdecken alte 50/52mm Rosetten
- **Einfache Montage:** Drehbewegung zum Arretieren
- **Griffel:** fest/drehbar gelagert

## Varianten

- **GK3:** Ohne sichtbare Schrauben (perfekte Optik)
- **GK4:** Mit Kugellager (Comfort-Version)
